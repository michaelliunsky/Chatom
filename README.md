# Chatom - 轻量 & 易部署的在线聊天室

**Chatom** 是一款极简、轻量的 PHP 单页面在线聊天室，支持**多人聊天、多房间、加密传输**，无需数据库，解压即用，适合快速搭建私密聊天环境。

## 主要功能

- **多人在线聊天** - 允许多人同时在线交流
- **多房间支持** - 可创建多个独立聊天房间
- **基础加密传输** - 采用 Base64 与字符替换方式保护信息隐私
- **轻量高效** - 使用 PHP 进行长连接读取，减少服务器负担
- **快速部署** - 只需 PHP 运行环境，无需数据库支持
- **聊天记录保存** - 聊天数据存储在 `chat_data` 文件夹中
- **自定义昵称** - 用户可设置个人昵称，浏览器会自动保存
- **私密房间** - 支持创建带密码保护的房间
- **自定义配置** - 通过 `config.php` 可设置网站标题、Logo 等参数

## 使用说明

1. **下载 Chatom** 并解压到服务器目录
2. 确保服务器支持 **PHP 运行环境**
3. 在 `config.php` 中配置网站标题和 Logo（可选）
4. 创建 `chat_data` 目录用于存储聊天记录
5. 访问`index.php`即可开始使用

## 关于 Chatom

**Chatom** 名称来源于：

- **Chat** - 代表聊天核心功能
- **Atom** - 象征轻量、极简和高效，如原子般小巧
